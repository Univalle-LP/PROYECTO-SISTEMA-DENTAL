Sistema Dental con Diagnóstico de Caries mediante IA
Descripción

Sistema web desarrollado en Django para la gestión de pacientes odontológicos, citas médicas, diagnósticos e imágenes dentales. El sistema incorpora un modelo de Inteligencia Artificial para la detección de caries a partir de imágenes radiográficas.

Objetivos
Gestionar pacientes odontológicos.
Administrar citas médicas.
Registrar diagnósticos.
Almacenar imágenes dentales.
Detectar caries mediante Inteligencia Artificial.
Generar información para apoyo clínico.

Instalación
Clonar el repositorio.
Crear entorno virtual.
Instalar dependencias:

pip install -r requirements.txt

Configurar PostgreSQL.
Ejecutar migraciones:

python manage.py migrate

Ejecutar el servidor:

python manage.py runserver
